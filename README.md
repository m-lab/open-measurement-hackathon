# Open Measurement Hackathons @ ACM IMC

Since 2022, M-Lab and other open measurement projects have hosted a hackathon at ACM's Internet Measurement Conference. 

* [2025 ACM IMC - Madison, Wisconsin, USA](2025/README.md)
* [2023 ACM IMC - Montreal, Canada](https://www.measurementlab.net/blog/imc-hackathon-2023/)
* [2022 ACM IMC - Nice, France](https://www.measurementlab.net/blog/imc22-hackathon/)
